# 🌟 Experiment: exp6_b

## Client-Side Form Validation

This project demonstrates **client-side form validation** using **HTML, CSS, and JavaScript**.
The form validates user input before submission to ensure correct data entry and improve user experience.

---

## 📌 Features

* ✨ Attractive modern UI with animations
* 🎨 Glassmorphism design with gradient background
* 📧 Email validation with domain checking
* 🔐 Password validation with multiple conditions
* 👁️ Show/Hide password toggle
* ✅ Success message on valid submission
* ⚡ Instant feedback without page reload

---

## 🧾 Input Fields

The form contains only **two fields**:

1. Email ID
2. Password

---

## ✔️ Validation Rules

### Email

* Must contain `@`
* Must contain a valid domain (`.com`, `.in`, or country code)

### Password

* Must start with a **capital letter**
* Must contain at least **one number**
* Must contain at least **one special character**
* Minimum **5 characters** required

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Client-Side Validation)

---

## 🚀 How to Run the Project

### Method 1 (Direct)

1. Download or clone the project.
2. Open the folder `exp6_b`.
3. Double-click `index.html`.

### Method 2 (VS Code Live Server)

1. Open the project folder in VS Code.
2. Install the **Live Server** extension.
3. Right-click `index.html` → **Open with Live Server**.
